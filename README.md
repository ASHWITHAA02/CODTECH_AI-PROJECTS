## 🚀 Projects Overview

### 1️⃣ Text Summarization

This project performs automatic summarization of large text documents using Natural Language Processing techniques.
The program extracts the most important sentences from a paragraph and generates a shorter summary.

**Technologies Used**

* Python
* NLTK (Natural Language Toolkit)

---

### 2️⃣ Speech Recognition

This task converts spoken audio into text using speech recognition techniques.

**Features**

* Reads an audio file
* Converts speech into readable text

**Technologies Used**

* Python
* SpeechRecognition Library

---

### 3️⃣ Image Style Transfer

This project demonstrates a basic image transformation where an input image is processed and transformed into a stylized output.

**Technologies Used**

* Python
* OpenCV / Image Processing

---

### 4️⃣ Text Generation

This project generates text automatically based on input prompts using simple Natural Language Processing techniques.

**Technologies Used**

* Python
* NLP concepts

## 🛠 Installation

Clone the repository:

```
git clone https://github.com/ASHWITHAA02/CODTECH_AI-PROJECTS.git
```

Navigate to the project folder:

```
cd CODTECH_AI-PROJECTS
```

Install required libraries:

```
pip install nltk
pip install SpeechRecognition
pip install opencv-python
```

---

## ▶️ How to Run the Projects

Example:

```
cd Task1_Text_Summarization
python summarizer.py
```

You can run each project individually inside its respective folder.
